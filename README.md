A cinematic, Apple-level portfolio built with raw HTML/CSS/JS — no frameworks, no build step, instant deploy. Features a **Three.js bokeh particle layer**, **GSAP entrance animations**, **talking-head video hero**, and fully interactive ML project demos.

## 🎬 Tech Stack

| Layer | Tech |
|-------|------|
| **3D Particles** | Three.js r128 — bokeh, additive blending, mouse parallax |
| **Animations** | GSAP 3.12 + ScrollTrigger — entrance, reveal, bar fill |
| **Video Hero** | HTML5 video — foreground + ambient blur duplicate |
| **Styling** | CSS custom properties, glassmorphism, cinematic dark theme |
| **Fonts** | Syne (headings) + Space Grotesk (body) + Lora (blog prose) |
| **Hosting** | Vercel (free) with optimised caching + video streaming headers |
| **Domain** | is-a.dev (free) → upgrade to .com whenever |
| **PWA** | Service Worker v2 with network-first strategy |

## 🚀 Deploy in 3 Steps

```bash
# 1. Clone and go
git clone https://github.com/Ve0602/ramana-portfolio.git
cd ramana-portfolio

# 2. Push to your GitHub repo
git remote set-url origin https://github.com/YOUR_USERNAME/ramana-portfolio.git
git push -u origin main

# 3. Deploy on Vercel
# → vercel.com → Import from GitHub → Deploy
# Zero config needed — vercel.json is already set
```

Then get your free domain: [domain-setup.html](./domain-setup.html)
