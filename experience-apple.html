What I Do

The 4 Pillars of This Role

Each area requires a distinct mental model. Here's how each one works.

Pillar 01
SFT Data Creation
Writing diverse, high-quality prompt-response pairs across coding, math (KaTeX), summarisation, rewriting, extraction, Q&A, tool use, and safety — following the LLM SFT Guideline v2.1.7. Every pair teaches the model a specific behaviour.
Pillar 02
Safety Taxonomy
Classifying every prompt across safety dimensions — Safe, Sensitive, Dual-use, Harmful. The same information can change category based on framing, intent, and context. Applied to every prompt written and evaluated.
Pillar 03
Multi-Turn Conversations
Designing complex conversations that demonstrate context retention, graceful topic transitions, clarification handling, and error recovery. Each turn must feel natural while teaching the model a specific multi-turn behaviour.
Pillar 04
Team Building Initiative
Independently identified coverage gaps in the prompt engineering team. Designed evaluation criteria, conducted interviews testing prompt quality and safety understanding, and onboarded new engineers — without being asked to.

What SFT Prompts Look Like

Real examples from the type of work I do — across different task categories. These are illustrative, not confidential Apple data.

sft_examples_ramana.jsonl
Prompt (User)
Summarise the following paragraph about climate feedback loops in exactly 2 sentences, using simple language suitable for a 10th-grade student. Do not include any information not present in the original. [PARAGRAPH]: Arctic permafrost contains vast amounts of frozen organic matter. As temperatures rise, this permafrost thaws and microbes begin decomposing the organic matter, releasing carbon dioxide and methane — potent greenhouse gases — which in turn cause further warming, creating a self-reinforcing cycle.
Format / Constraints
Exactly 2 sentences Simple language No hallucination Grade 10
Response (Assistant)
When temperatures rise, frozen ground called permafrost starts to melt, which lets tiny organisms break down the stored material and release warming gases like carbon dioxide and methane. These gases then make the Earth even warmer, which causes more permafrost to melt — a cycle that keeps feeding itself.
Quality Signals
✓ Exactly 2 sentences — constraint followed precisely
✓ No jargon — "permafrost" explained inline
✓ No added info — strictly from source paragraph
✓ Feedback loop clearly conveyed
