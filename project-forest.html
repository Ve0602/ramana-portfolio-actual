01
Algorithm
Canadian FWI System Implementation
Implemented the complete Canadian Forest Weather Index system from scratch: FFMC (Fine Fuel Moisture Code), DMC (Duff Moisture Code), DC (Drought Code), ISI (Initial Spread Index), BUI (Buildup Index), and the final FWI score.
def calc_ffmc(temp, humidity, wind, rain, prev_ffmc=85.0): mo = 147.2*(101-prev_ffmc)/(59.5+prev_ffmc) # Rain correction, drying rate, wetting rate... return 59.5*(250-m)/(147.2+m) def calc_fwi(isi, bui): if bui <= 80: fd = 0.626*(bui**0.809)+2 else: fd = 1000/(25+108.64*math.exp(-0.023*bui)) return math.exp(2.72*(0.434*math.log(isi*fd))**0.647)
